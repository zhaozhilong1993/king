Welcome to king's documentation!
================================
King is an OpenStack billing program.It supports different ways of billing, including time and usage.
King also has its own internal self-test system to ensure the accuracy of billing.


King's purpose and vision
=========================

* King provides the basic framework, allowing users to customize the price.
* Users do not need to consider too much internal call logic.
* King provides a REST API that covers all operations.
* King can efficiently handle large amounts of consumer data.


This documentation offers information aimed at end-users, operators and
developers of King.
